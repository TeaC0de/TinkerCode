Mastering CSS Basics – Styling My Blog 🌈

What is CSS and Why It Matters? 🤔

CSS (Cascading Style Sheets) is what makes websites beautiful. While HTML structures the content, CSS styles it, adding color, fonts, and layout. It’s like dressing up your website and giving it personality!

Quick Tutorial – How I Added CSS to My Blog 🖌️

1. Link Your CSS File: Add this line in the <head> of your HTML:

<link rel="stylesheet" href="/TinkerCode/CSS/styles.css">

2. Add Basic Styling: Here’s a snippet from my styles.css:

body {
    font-family: 'Arial', sans-serif;
    background-color: #f0f0f0;
    color: #333;
    margin: 20px;
}

header {
    text-align: center;
    background-color: #c7d3f3;
    padding: 10px 0;
}

The Box Model – The Key to Layout 📦

Every element on a page is a box. To style it properly, use padding, border, and margin:

article {
    padding: 15px;
    border: 2px solid #333;
    margin: 20px 0;
    background-color: #ffffff;
}

What’s Flexbox and Why Should You Care? 🌟

Flexbox is a CSS layout module that helps arrange items in a row or column with flexible sizing. It’s perfect for building responsive layouts! Here’s a simple flexbox snippet:

nav ul {
    display: flex;
    justify-content: center;
}

This code ensures your navigation links are evenly spaced in a row. I’m excited to explore flexbox in more depth in future updates!
